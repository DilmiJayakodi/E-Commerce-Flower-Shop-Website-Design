🚀 About The Project

This repository contains the source code for a modern flower shop landing page. The design is elegant and organic, utilizing a soft color palette to reflect the nature of the business. It serves as a great template for florists, garden centers, or anyone looking to build a lightweight e-commerce presence.

The main goal of this project was to demonstrate how far pure CSS can go in creating professional, responsive layouts without relying on heavy libraries.
✨ Key Features

    🚫 Zero JavaScript: Fully functional layout using only HTML & CSS.
    📱 Fully Responsive: Adapts perfectly to desktops, tablets, and mobile devices.
    🎨 Modern Aesthetics: Clean typography, hover effects, and organized layout structure.
    ⚡ Lightweight: Fast loading times with minimal file sizes.
    🛠 Easy to Customize: Clean code structure makes it easy to change colors, images, and text.

🛠 Technologies Used

    HTML5: For semantic structuring of the content.
    CSS3: For styling, Flexbox/Grid layouts, media queries, and transitions.
